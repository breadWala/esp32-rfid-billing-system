# ESP32 RFID Billing System

## Overview
This project is an ESP32 + RC522 RFID-based billing system developed using ESP-IDF. The system scans RFID tags and sends billing data through HTTP communication to display results on a mobile browser in real time.

## Features
- RFID tag detection
- ESP32-based embedded system
- HTTP communication
- Real-time browser display
- Modular architecture
- MQTT integration in progress

## Hardware Used
- ESP32
- RC522 RFID module

## Software/Frameworks
- ESP-IDF
- Embedded C
- HTTP protocol

## Communication Flow
RC522 → ESP32 → HTTP Request → Browser Display

## Future Improvements
- MQTT-based communication
- Backend database integration
- Multi-client synchronization
